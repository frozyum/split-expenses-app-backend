### Split Expenses App Backend
App for splitting expenses

### How to run the project:
1. Install dependencies: `pip install -r requirements.txt`
2. Start server: `python manage.py runserver`

### Technologies used:
* Python 3
* Django
* Docker/Docker-Compose
